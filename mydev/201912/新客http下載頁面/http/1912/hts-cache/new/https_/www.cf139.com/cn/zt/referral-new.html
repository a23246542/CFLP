  3. 3.推荐有效期为 90 天(被推荐人成功入金激活账户起 90 天内)，如被推荐人未能在此有效期内完成规定的交易手数，该推荐将被视为无效；交易返利以平仓单计算，期间开仓但未能完成平仓的订单将不计入统计中；
